# 🌌 Space Tourism

A simple interactive space-themed web project built with HTML, CSS, and JavaScript.

---

## 🚀 Features
- Planet selection (Moon, Mars, Earth, Jupiter)
- Smooth UI transitions
- Animated counters (distance & travel time)
- Canvas-based starfield background
- Responsive design for mobile & desktop
- Modal planet preview

---

## 📁 Project Structure

index.html
style.css
script.js
assets/


---

## ▶️ How to Run
Open `index.html` in your browser.

---

## 🛠️ Built With
- HTML
- CSS
- JavaScript (Vanilla)
- Canvas API
